# Example Plugin

A simple example plugin demonstrating skill structure and best practices for Claude Code.

## Skills

### `/hello`
A simple greeting skill demonstrating proper structure.
- **Usage:** `/hello`
- **Output:** Friendly greeting message

## Installation

From Claude Code CLI:
```
/plugin marketplace add https://github.com/EmilMachine/skillhub
/plugin install example-plugin
```

## Features

- Demonstrates YAML frontmatter
- Shows terse implementation style
- Provides concrete examples
- Follows md3step format conventions
