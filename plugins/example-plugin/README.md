# Example Plugin

A simple example plugin demonstrating the structure and best practices for Claude Code skills.

## Description

This plugin provides a basic "hello" skill that demonstrates proper skill structure, including YAML frontmatter, clear documentation, and usage examples.

## Installation

If you've already installed the skillhub marketplace:

```bash
claude plugin install skillhub/example-plugin
```

## Skills Included

### hello

A simple greeting skill that responds with a friendly message.

**Usage:**
```
/hello
```

**What it does:**
- Responds with a friendly greeting
- Demonstrates basic skill structure
- Shows proper YAML frontmatter usage

## Prerequisites

- Claude Code CLI installed
- skillhub marketplace added to your Claude plugins
