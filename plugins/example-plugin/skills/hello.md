# Hello Skill

## What it does

This skill provides a friendly greeting and demonstrates the basic structure of a Claude Code skill. It's meant to serve as a reference example for creating new skills.

## Usage

Invoke this skill using:

```
/hello
```

## Examples

**Basic usage:**
```
User: /hello
Claude: Hello! This is an example skill from the skillhub marketplace.
        I'm here to demonstrate proper skill structure and documentation.
```

## Implementation

When invoked, respond with a friendly greeting that explains this is an example skill. Keep the response concise and helpful.

Key elements demonstrated:
- YAML frontmatter with name and description
- Clear "What it does" section
- Usage instructions
- Concrete examples
- Implementation guidance

## Notes

This is a minimal example to show structure. Real skills should provide more detailed implementation instructions and handle various user scenarios.
